# NovaShop — interfață magazin online

Acest proiect conține o interfață HTML + CSS pentru un magazin online cu produse și coș de cumpărături.

## Structura proiectului

- `index.html` — pagina principală;
- `css/style.css` — stilizarea interfeței;
- `js/app.js` — fișier pregătit pentru funcționalități JavaScript;
- `img/` — dosar rezervat pentru imagini.

## Sarcina elevilor

Elevii trebuie să implementeze în `js/app.js` următoarele funcționalități:

1. Adăugarea produsului în coș.
2. Creșterea cantității dacă produsul există deja în coș.
3. Afișarea produselor în coș.
4. Calcularea subtotalului și totalului.
5. Actualizarea numărului de produse din coș.
6. Ștergerea produsului din coș.
7. Modificarea cantității produsului.
8. Ascunderea mesajului „Coșul este gol” când există produse.

## Observație

Proiectul nu conține funcționalități JavaScript implementate. Acestea sunt lăsate intenționat ca exercițiu practic.
